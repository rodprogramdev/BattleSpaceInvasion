## 📌 Overview
**Battle Space Invasion** is a 2D space shooter developed in Unity using C#.  
The game features dynamic combat systems, player transformation mechanics, enemy encounters, and multi-level progression.

Developed from **February – July 2022** by an 11-member team under *HomeTeam GameDev*.
